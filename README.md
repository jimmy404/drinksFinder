## Installation 

Install drinksFinder with **npm**

```bash 
  1) Clone project: drinksFinder (url)
  2) cd drinksFinder/
  3) npm install
  4) npm start
```

Install drinksFinder with **yarn**

```bash 
  1) Clone project: drinksFinder (url)
  2) cd drinksFinder/
  3) yarn install
  4) yarn start
```
